# css-mask
Demo for CSS mask

## Honda Civic demo with css mask
• Hero image: css mask vs `svg path`

• Author avatar image masked with circular png with alpha gradient around edges

• Use of `::first-letter` pseudo element

http://katomonster-css-mask-honda-civic.surge.sh/

## Nissan Leaf demo with css mask
• Hero image with css mask, which has slight alpha gradient around edges.

• Triangle shapes the `h3` header with `shape-outside`

• Author avatar image clipped with `clip-path`

• Text flows around inset image using `shap-outside`

http://katomonster-css-mask-nissan-leaf.surge.sh/

• Use of `::first-letter` pseudo element
